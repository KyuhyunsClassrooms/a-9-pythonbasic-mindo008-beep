# AI 기초를 위한 파이썬 읽기 수업

이 저장소는 **인공지능 기초 수업을 듣기 전에 필요한 파이썬 문법을 읽고 이해하는 3차시 notebook 수업용 템플릿**입니다.  
이번 수업의 목표는 **문제를 많이 푸는 것**이 아니라, **코드를 읽고 설명하고 한 줄 수정할 수 있는 학생**을 만드는 것입니다.

## 이 저장소에서 사용할 파일 순서

1. `01_notebook_variables.ipynb`
2. `02_list_loop_if.ipynb`
3. `03_ai_code_reading.ipynb`

## 수업에서 지킬 규칙

- 셀은 **위에서 아래로** 실행합니다.
- 코드를 실행하기 전에 **출력 결과를 먼저 예측**합니다.
- 한 번에 한 줄만 바꿔 보고, 결과가 왜 달라졌는지 말로 설명합니다.
- 막히면 바로 답만 복사하지 말고, **어느 줄이 이해되지 않는지**부터 적습니다.

## 학생용 실행 방법

### GitHub Codespaces를 쓰는 경우
1. 과제를 수락한 뒤 저장소를 엽니다.
2. **Code → Codespaces**에서 새 Codespace를 엽니다.
3. 왼쪽 탐색기에서 오늘의 notebook 파일을 엽니다.
4. 셀을 클릭하고 **Shift + Enter**로 실행합니다.

### VS Code를 쓰는 경우
1. 저장소를 VS Code로 엽니다.
2. Python / Jupyter 확장이 설치되어 있는지 확인합니다.
3. 오늘의 notebook 파일을 열고 **Shift + Enter**로 실행합니다.
